# UX Flows

## Kanban
- **Add card**: Click “Add task” → modal → select list → create.
- **Move card**: Drag card to column → snap → persist.
- **Delete card**: Trash icon removes card from column.

## Jobs
- **Save job**: click Save → status becomes “saved”.
- **Apply job**: click Apply → status becomes “applied”.
- **Filter**: status chips filter table.

## Repos
- **Open PR**: click Open → navigate to PR detail link.
- **Filter status**: chips for green/yellow/red.

## Settings
- **Connect integration**: click Connect → OAuth flow.
- **Toggle notifications**: enable/disable per channel.
